Welcome to hSlides!

Hi there! Welcome to hSlides!

hSlides is a HTML-based slides showing platform. Indeed, it is free, and it is cross-platform. You will not worry about if the computers in presentation halls are running with slides showing application, e.g. MS Powerpoint and Apple iWork, because almost very single computer has an installation of web browser.

In terms of fabricating slides, hSlides is a WYTIWYG (What You Think Is What You Get) platform. Just like LaTeX, the slides are being prepared by using text-based codes. More precisely, slides for hSlides are using HTML.

Construct a slide

All slides are structured by using HTML5 tag "<section> ... </section>".

The structure of each slides show should look like:

<section>
1st slide
</section>
<section>
2nd slide
</section>
<section>
3rd slide
</section>
...
